# Short Video Feed

A short-video feed application inspired by TikTok, built with Next.js, TypeScript, and TailwindCSS.

## Features

- Vertical video feed with CSS Scroll Snap.
- Auto-play active video.
- Auto-pause inactive videos.
- Audio control (mute/unmute and volume adjustment).
- Video and audio synchronization.
- Like / Unlike functionality.
- Global state management using React Context API.
- Responsive

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- TailwindCSS v4
- React Context API
- React Icons

## Installation

```bash
npm install
npm run dev
```
